The RUSA web site does not provide a programmatic interface, but the web pages are simple and easy to parse. With minimum effort, it is possible to transform the data is JSON format.

Retrieve Club members data

Save the page with the list of users. Once the page is saved, a command such as: 

grep "resultsearch_PF.pl" rmcc.html  | awk -F "mid=" '{print $2}'
> | awk -F "&sortby=" '{print $1}' > ids.txt
